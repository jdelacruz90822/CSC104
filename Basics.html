CS014 Basic HTML

HTML stands for Hypertext Markup Language. HTML provides structure of the page by marking up the text. You can use a variety of HTML tags to structure and organize a webpage. HTML does not really have any rules, but there are best practices that should be used when building the webpage.

History of HTML

  • HTML 1.0 - developed in the early 1990's
  • HTML 2.0 released in 1995
  • HTML 3.2 - released in 1997
  • HTML 4.0 - released in 1999
  • HTML 5 - current version released in 2014 - living standard

Cascading Stle Sheets

CSS is a language that is used to make HTML look better. CSS can control the layouts, colors, fonts, basically any visual element on the webpage can be changed from the default with CSS

History of CSS

  • CSS1 released in 1996
  • CSS2 was released in 1998
  • CSS3 - current version - released in 2011- living standard

Basic structure

    Selector{
        Preperty; value; 
    }

CSS Selectors

  • HTML eletment
  • id of an HTML element - #id
  • Custom class - .classname

3 Ways to create CSS

  • External - a seperate CSS file - example, main.css
  • Embedded- style tags in the head section
  • Inline - we set up css on individual HTML elements
